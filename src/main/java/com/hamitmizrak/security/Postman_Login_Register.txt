Swagger Bearer Token Açmak
Swagger UI’da token nasıl girilir?

Önce /auth/login veya sizin login endpoint’inizden JWT alın (response body’de accessToken, token, jwt vb.).

Tarayıcıda /swagger (veya /swagger-ui.html) açın.
Üstte Authorize butonuna tıklayın.
Açılan pencerede yalnızca raw JWT’yi yapıştırın (örn. eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...).

“Bearer ” yazmayın. Şemayı scheme=bearer yaptığımız için Swagger UI otomatik Authorization: Bearer <token> header’ını oluşturur.

Authorize → Close.
Artık tüm korunan endpoint çağrılarında header otomatik eklenecek.
Çıkış yapmak için tekrar Authorize butonuna basıp Logout deyin.
